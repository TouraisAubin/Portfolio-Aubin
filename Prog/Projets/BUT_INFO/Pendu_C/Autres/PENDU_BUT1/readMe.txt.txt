Bienvenue dans le jeu du Pendu de l'équipe 'Ekip'


I. INTRODUCTION :

Il s'agit d'un jeu individuel avec un total de 20 mots à deviner, classés dans 4 thèmes au choix (5 mots par thèmes).
Toutes les règles sont précisées dans le jeu, veuillez bien les suivre pour éviter tous problèmes ou confusion avant le démarrage du jeu. 

Dans ce dossier que vous avez décompressé, où vous m'avez trouvé, vous avez accès pourrez retrouver :
- Le jeu en lui même, s'intitulant "JEU_PENDU_EKIP.c" (code source) ou "JEU_PENDU_EKIP.exe" (l'éxecutable)
- Toutes les algorithmes des fonctions C qui se trouveront dans le jeu en question
- Toutes les fonctions en C
- Un topo sur les fonctions de notre jeu, pouvant ainsi comprendre à quoi sert nos fonctions 
- Le cahier des charges de notre projet


II. MISE EN PLACE :

Pour pouvoir jouer à notre jeu, vous aurez besoin d'un compilateur de fichier .c, ou bien d'un éditeur de texte étant capable traiter cela comme peut le Code bloc ou Visual Studio (pouvant être configuré de façon à avoir un compilateur intégré)
Et evidement de quoi saisir du texte pour jouer.

Pour pouvoir jouer au jeu, il vous faudra ouvrir le fichier "JEU_PENDU_EKIP.exe" ou bien compiler le fichier source "JEU_PENDU_EKIP.c" et l'éxecuter par la suite.
Pour que le jeu se lance et pour que toutes les fonctions se lancent correctement, le jeu aura besoin de tous les fichiers annexe ainsi que de toutes ses fonctions, mis à part les documents "Enoncé" qui sont : le cahier des charges et le topo que les fonctions.

Pour installer le jeu, il vous suffit de cliquer sur "JEU_PENDU_EKIP.exe" sans le retirer de son doisier d'origine. 
Vous avez aussi la possibilité de compiler le "JEU_PENDU_EKIP.c" à l'aide d'un compilateur depuis votre terminal à l'aide de par exemple Mingw, que vous pouvez installer en ligne : https://winlibs.com

Ou encore même d'un compilateur en ligne, comme par exemple : https://www.onlinegdb.com/online_c_compiler
Vous aurez juste à faire les étapes suivantes :
- Ouvrir "JEU_PENDU_EKIP.c" dans un éditeurs texte (comme WordPad, SublimeTexte etc.)
- Faire le raccourci CTRL + A (pour tout sélectionner)
- Faire CTRL + C (pour copier la totalité du code)
- Aller sur le lien du compilateur donné auparavant
- Faire CTRL + A, puis SUPPR (pour vidé le cache sur possède le compilateur au démarrage)
- Faire CTRL + V (pour coller le programme du jeu "JEU_PENDU_EKIP"
- Appuyez sur le bouton RUN afin de jouer au jeu
