ROBOT NIRYO

Le Sujet Les Étapes de Réalisation Conclusion

Le Sujet

L’objectif du projet Robot Niryo est de concevoir un robot autonome capable d’effectuer une tâche de "Pick and Place" (prise et dépose d'objets).

Le robot doit être capable de détecter un objet, planifier son mouvement, le saisir, le déplacer, puis le déposer à un emplacement défini.

Le projet utilise un bras robotique avec des capteurs pour détecter les objets, une planification de mouvement pour guider le robot et un système de préhension pour manipuler l’objet.

Les Étapes de Réalisation

Étape 1 : Organigramme

Objectif :
Représenter visuellement le flux d’actions que le robot doit suivre pour accomplir sa tâche de "Pick and Place".

Description :
Le diagramme de flux suit les étapes suivantes :
1. Détection de l’objet.
2. Planification du mouvement.
3. Saisie de l’objet.
4. Déplacement vers la zone de dépôt.
5. Dépôt de l’objet.
6. Retour à la position initiale.

Chaque étape du processus est illustrée dans le diagramme de flux pour clarifier les différentes actions à réaliser.

Étape 2 : Workspace

Objectif :
Créer un environnement de travail pour le robot, dans lequel il pourra interagir avec les objets à manipuler.

Description :
Le workspace est une zone délimitée où le robot évolue pour détecter et manipuler les objets.

Des capteurs de proximité et une caméra sont utilisés pour repérer les objets dans l'espace et définir leurs positions.

Le robot doit pouvoir se déplacer dans cette zone de manière fluide et éviter les obstacles.

Étape 3 : Programmation en Bloc

Objectif :
Utiliser la programmation par blocs pour développer le code qui gère les actions du robot.

Description :
La programmation par blocs permet de créer des séquences logiques pour contrôler le robot.

Chaque action (comme la détection d'objet, le mouvement, la prise et la dépose) est programmée en blocs distincts, facilitant ainsi la gestion et la modification du programme.

Le robot suit des instructions claires à chaque étape, du début à la fin du cycle "Pick and Place".

Conclusion du Projet

En conclusion, le projet Robot Niryo met en œuvre un robot autonome capable de réaliser des tâches complexes de prise et de dépose d'objets avec une grande précision.

Grâce à des capteurs de détection, une planification de mouvement efficace et une programmation par blocs, ce robot est capable de manipuler des objets dans un environnement défini avec succès.

Ce projet démontre l'importance de l'automatisation dans les processus industriels et ouvre la voie à de nombreuses applications dans différents domaines.
